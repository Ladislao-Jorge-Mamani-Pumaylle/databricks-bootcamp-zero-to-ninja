# databricks-bootcamp-zero-to-ninja

Este bootcamp está diseñado para aprender Apache Spark desde cero, utilizando Databricks como plataforma principal. A lo largo del curso, se cubren los conceptos fundamentales y avanzados de Spark, permitiendo a los participantes adquirir habilidades prácticas y teóricas para trabajar con grandes volúmenes de datos de manera eficiente.
